# Mandarin Study Guide

Interactive single-page study guide generated from a personal Mandarin glossary:
74 terms across 7 topics, with tone-colored pinyin, usage notes, example sentences,
character breakdowns, live search/filter/sort, and a flashcard study mode.

`index.html` is fully self-contained — open it directly, or deploy as a static site
(zero config). Deployed on Vercel via Git integration.
